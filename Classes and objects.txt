Classes 
    Khác với variable và defination name bắt đầu bằng chữ in thường, thì class name bắt đầu bằng chữ in hoa.
    Dùng để tái sử dụng code, chỉ cần 1 logic sử dụng cho nhiều đối tượng
    Dùng để quản lý dễ dàng, gom nhóm dữ liệu liên quan, code gọn, dễ bảo trì.

Class có 2 phần chính
    Thuộc tính (Attributes): gồm thông tin, đặc điểm của đối tượng (tên, tuổi, màu sắc)
    Đối tượng (Methods): Hành động mà đối tượng có thể thực hiện(chạy, nói, tính toán)

Khái niệm quan trọng:
    self: nó đại diện cho chính đối tượng được gọi, luôn luôn có self
    __init__: khi gọi class thì __init__ sẽ tự động được gọi để thiết lập giá trị trong hàm __init__ cho đối tượng.

Example:

    class Dog:
        def __init__(self, name):
            self.name = name
            self.legs = 4

        def speak(self):
            print(self.name + ' says: Bark!')

    my_dog = Dog('Rover')
    another_dog = Dog('Fluffy')

    my_dog.speak()
    >>>>    Rover says: Bark!

    another_dog.speak()
    >>>>    Fluffy says: Bark!
